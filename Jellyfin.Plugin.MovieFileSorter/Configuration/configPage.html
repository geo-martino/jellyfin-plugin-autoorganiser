Unchecking this will only place a movie file in a sub-folder if it has movie extras. Movies without extras will then be placed in the root folder.

Label settings:

Configure a label to add to the file name e.g. Dune (2021) - [1080p HEVC].
Label parts are arranged in the same order seen here.

e.g. 480p, 576p 720p, 1080p, 2160p
e.g. HEVC, H264, AV1
e.g. 8bit, 10bit
